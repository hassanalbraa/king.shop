# **App Name**: KING STORE

## Core Features:

- User Authentication: Secure user login with username and password. Differentiate between standard user and admin access.
- User Dashboard: Display user-specific information upon login, including welcome message, current balance, and available offers (game cards).
- Offer Management: Present game offers as interactive cards. Display cards clearly with hover effects. Allows users to browse available game cards and purchase them using their balance.
- Admin Dashboard: Dedicated admin interface for user management, balance adjustments, and offer price modifications.
- Settings Management: Allow standard users to modify their password, ensuring secure account management.

## Style Guidelines:

- Primary color: Gradient from light blue (#007BFF) to cyan (#00C3FF) for the top title bar.
- Background color: Very light blue (#F0F4FF) to provide a clean and non-distracting backdrop.
- Accent color: Darker blue (#0056b3) for button hover effects and borders on active game cards.
- Font: 'Poppins' (sans-serif) for a clear and modern aesthetic. Note: currently only Google Fonts are supported.
- Centralized white container with rounded corners and soft shadows to host login, user dashboard, and admin interface elements, ensuring responsiveness on both mobile and desktop devices.
- Right-to-Left (RTL) text direction for Arabic language support.
- Subtle hover effects on buttons and game cards to enhance user interaction.
